## ADDED Requirements

### Requirement: Hybrid Planning Prompt Template
The hybrid-planning-template SHALL generate structure plans for novels combining historical and xuanhuan elements.

#### Scenario: Act planning with dual-layer structure
- **WHEN** generating act planning for a hybrid novel
- **THEN** the prompt emphasizes historical surface narrative (dynasty/politics) with xuanhuan underlying mechanics (cultivation/martial arts)

#### Scenario: Chapter outline with dual satisfactions
- **WHEN** generating chapter outlines for a hybrid novel
- **THEN** the prompt includes guidance on delivering both historical authenticity and xuanhuan excitement in each chapter

#### Scenario: World-building consistency
- **WHEN** planning story structure for a hybrid novel
- **THEN** the prompt guides to ensure cultivation system aligns with historical period setting

### Requirement: Hybrid Novel Tone
The hybrid planning prompt SHALL use language emphasizing both historical texture and cultivation excitement.

#### Scenario: Tone consistency
- **WHEN** generating hybrid planning content
- **THEN** the system prompt describes the writer as "精通历史与玄幻融合叙事，兼顾权谋逻辑和修炼体系"
